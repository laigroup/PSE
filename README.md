# PSE

## Prerequisites

```
sudo apt-get install build-essential cmake
sudo apt-get install zlib1g.dev
sudo apt-get install libgmp-dev
```

## Commands

```
mkdir build && cd build
cp ../scripts/build.sh .
chmod u+x build.sh
./build.sh  
```

## Usage

### Using Binary

Use "PSE --help" to see the usage of the tool.
